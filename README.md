# CS321-Project - "Strength"

## Overview
Strength is a web application designed to provide avid fitness users with a product that tracks a user's workouts and customize it towards a user’s goals. It will process the user’s goals and customize the workout plan through the exercise selection, intensity/weight (RPE), and volume of the workout. It will also provide a real time exercise playback, so users will keep their form accountable. Strength provides a solution to simplify a user’s journaling experience, so they can be held accountable to attain their fitness goals.
